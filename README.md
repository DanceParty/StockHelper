## To run:

```
Git clone https://github.com/jkd65/StockHelper.git
cd stockhelper
meteor npm install
meteor run
```

## Tools

* MeteorJS 1.5
* BlazeJS
* [Clearbit Company Logo API](http://blog.clearbit.com/logo/)
* MongoDB
